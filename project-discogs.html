Contexte

Passion pour la musique et les vinyles — et une question : est-ce que le format physique a une valeur objective sur le marché secondaire ? Discogs, la plus grande marketplace de vinyles, contenait la réponse dans ses données.

Objectif

Scraper Discogs via Python pour constituer un dataset sur les prix, formats et genres musicaux — puis analyser les corrélations entre format physique, rareté et valeur perçue sur le marché.

Résultat

Dataset constitué, analyse de corrélation complète — insights sur les formats les plus valorisés, les genres les plus côtés et les patterns de prix sur le marché du vinyle d'occasion.
