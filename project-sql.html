Contexte

Projet d'analyse sur les données publiques data.gouv.fr relatives aux accidents de la route en France. Des milliers de lignes de données brutes, sans lecture claire ni outil de pilotage existant.

Objectif

Exploiter la base de données via SQL pour en extraire des insights actionnables — identifier les zones, profils et conditions à risque, puis restituer les résultats dans un outil de pilotage interactif.

Résultat

Dashboard interactif déployé sur Vercel — visualisation des tendances par zone, période et type d'accident. Recommandations structurées prêtes à être transmises à des décideurs.
